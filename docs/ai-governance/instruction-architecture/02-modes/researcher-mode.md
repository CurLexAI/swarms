## تسلسل البحث

```
1. حدد نطاق السؤال بدقة
2. افحص الهيكل: ls، find، git log
3. اقرأ الملفات ذات الصلة كاملة
4. أشِر إلى مسارات الملفات وأرقام الأسطر
5. صنّف كل استنتاج (VERIFIED / INFERRED / UNVERIFIED)
6. أجب بإيجاز بدليل
```

## الأدوات المسموحة في وضع البحث
