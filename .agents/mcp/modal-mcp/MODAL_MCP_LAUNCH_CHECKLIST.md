# MODAL_MCP_LAUNCH_CHECKLIST

## قبل التشغيل
- [ ] ملف `.env` موجود ومعبأ بالقيم المطلوبة
- [ ] صلاحيات **Modal API token** صحيحة
- [ ] لا توجد أسرار داخل `README` أو الكود أو logs

## التحقق المحلي
- [ ] `npm install`
- [ ] `npm run build`
- [ ] تشغيل السيرفر محليًا بدون أخطاء
- [ ] endpoint الـ MCP يفتح على الرابط المتوقع

## التحقق الوظيفي
- [ ] **List Tools** يعرض الأدوات المتوقعة فقط
- [ ] كل أداة ترجع response صحيح مع input صحيح
- [ ] الأخطاء مفهومة عند input ناقص أو token غير صالح
- [ ] لا توجد أدوات زائدة أو تجريبية مكشوفة

## التحقق الأمني
- [ ] السيرفر **read-only** فعلًا إذا هذا هو المقصود الآن
- [ ] لا يوجد تنفيذ deploy/restart/write بالخطأ
- [ ] الأسرار لا تظهر في الرسائل أو المخرجات
- [ ] الأدوات لا تُرجع بيانات حساسة أكثر من اللازم

## MCP Inspector
- [ ] `List Tools` ناجح
- [ ] `Call Tool` ناجح على الأقل لأهم الأدوات
- [ ] responses منظمة وثابتة
- [ ] لا توجد console errors

## داخل ChatGPT
- [ ] ChatGPT يختار أداة Modal فقط عندما يكون الطلب متعلقًا بـ Modal
- [ ] لا يستدعي Modal في prompts غير المرتبطة بالتشغيل
- [ ] الوسائط المرسلة للأداة صحيحة
- [ ] السلوك السلبي والحدودي تم اختباره

## حكم الجاهزية
اعتبر `modal-mcp` **جاهزًا مبدئيًا** فقط إذا:
- [ ] build نجح
- [ ] التشغيل المحلي نجح
- [ ] Inspector نجح
- [ ] اختيار الأداة داخل ChatGPT كان صحيحًا
- [ ] لا توجد ثغرات واضحة في read-only boundaries
