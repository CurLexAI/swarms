# Qarar Swarms Sovereign Integration Overlay

This bundle adds a safe additive integration layer between Swarms and Qarar sovereign models.

Run from repository root:

```bash
bash qarar-swarms-sovereign-integration/install_swarms_patch.sh .
```

The overlay assumes the Bayyinah scaffold exists at `qarar/packages/bayyinah`. If it does not exist, run the recovery PR executor first.
